# Customer Churn Prediction Dashboard

## Project Overview

This project predicts customer churn using Machine Learning and displays insights through a Streamlit dashboard.

## Dataset

Telco Customer Churn Dataset

- Total Customers: 7043
- Features Used: 30

## Machine Learning Model

- Logistic Regression
- Accuracy: 80.38%
- ROC-AUC: 0.836

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Plotly
- Streamlit

## Dashboard Features

- Customer Churn Prediction
- Risk Probability Score
- Feature Importance Analysis
- Analytics Dashboard
- Business Insights
